Project Overview: The VayuSetu Airline Booking System is a console-based application developed using the C programming language. This project simulates the essential operations of an airline reservation system, allowing users to browse flights, check seat availability, book seats with an auto-generated PNR, cancel bookings, compare sample fares, and access a complete action history. It also includes an optional admin panel for managing and updating flight details, and follows a modular programming structure using multiple .c and .h files.

It allow to:

1:- View and explore all available flights.  
2:- Check real-time seat availability.  
3:- Book seats with automatic 8-digit PNR generation.  
4:- Cancel previously booked seats.  
5:- Compare mock flight fares for better decision-making.  
6:- View a session-based activity and action history.  
7:- Use admin controls to add or update flight information.  
8:- Maintain a clean modular structure using header files and multiple source files.  

Features:

1:- Menu-driven console interface.  
2:- Real-time flight and seat availability display.  
3:- Automatic PNR generation for each booking.  
4:- Seat cancellation with updated availability tracking.  
5:- Rate comparison module for mock fares.  
6:- User activity history recorder.  
7:- Optional admin flight management panel.  
8:- Modular architecture with file handling support for scalability.
